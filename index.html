Welcome to the Text RPG

Hi there, welcome to the overrun Animal Earth!

You must rescue animal earth from humans. Choose a weapon: Mace, Bow, or Sword.

To play the game, follow the instructions below to download and run it on your local machine.

Instructions

  1. Download the game source code from GitHub.
  2. Make sure you have Python installed.
  3. Run the game by executing python app.py in your terminal.
